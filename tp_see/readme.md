# TP MQTT

## Configuration du broker

copier le fichier de configuration, puis insérer les paramètres correspondants à votre configuration:

```bash
cp client_c/config_dist.h client_c/config.h
```

Configurer les paramètres:
- adresse broker
- utilisateur
- mot de passe
